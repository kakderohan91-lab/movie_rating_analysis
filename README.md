# Movie Ratings Analysis 🎬

This project analyzes movie rating data using Python and data visualization techniques.
The goal is to explore movie ratings, identify trends, and understand which movies and genres perform the best.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Tasks

* Clean and preprocess the movie dataset
* Calculate summary statistics such as mean, median, and mode of ratings
* Visualize rating distributions using histograms and box plots
* Identify top-rated movies and popular genres
* Analyze relationships between ratings, revenue, and other variables

## Key Insights

* Most movies have ratings between **6 and 7.5**
* Some genres such as **Drama and Action** appear more frequently
* Higher rated movies often receive more audience engagement
